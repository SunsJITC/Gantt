# Gantt# 📋 周计划甘特图

一个轻量、美观的每周任务甘特图打卡工具。纯前端单 HTML 文件，无需安装任何依赖，双击即可使用。



---

## ✨ 功能特性

- **甘特图视图** — 按板块、任务、日期展示整周计划，一目了然
- **点击打卡** — 点击色条标记完成，附带撒花动画反馈
- **实时进度** — 顶部总进度条 + 底部每日完成率统计
- **前端编辑** — 点击「✏️ 编辑计划」即可在页面上直接修改，无需改代码：
  - 设置起始日期和天数（5–14 天）
  - 增删板块，自定义名称、emoji、颜色
  - 增删任务，逐日勾选排期
- **自动保存** — 计划配置和打卡记录分别存储在 localStorage，刷新不丢失
- **零依赖** — 纯原生 HTML + CSS + JavaScript，单文件约 20KB
- **深色主题** — 默认暗色界面，长时间使用护眼
- **移动适配** — 响应式布局，支持 iOS Safe Area

## 🚀 使用方式

### 最简单：直接打开

双击 `周计划甘特图.html`，在浏览器中即可使用。

### 添加到手机主屏幕（PWA 体验）

需要先将文件托管到一个 HTTPS 地址，然后通过 Safari/Chrome 添加到主屏幕。

**1. 托管文件（任选一种）**

| 方式 | 操作 |
|------|------|
| **GitHub Pages** | 上传到 GitHub 仓库 → Settings → Pages → 选 main 分支 |
| **Vercel** | 拖入文件夹到 vercel.com 即可 |
| **Netlify** | 拖入文件夹到 netlify.com 即可 |
| **本地预览** | `python3 -m http.server 8080` 然后访问 `localhost:8080` |

**2. 添加到主屏幕**

| 平台 | 操作 |
|------|------|
| **iPhone / iPad** | Safari 打开 → 底部分享按钮 ↑ → 「添加到主屏幕」 |
| **macOS Sonoma+** | Safari 打开 → 文件 → 「添加到程序坞」 |
| **Android** | Chrome 打开 → 右上角 ⋮ → 「添加到主屏幕」 |

## ✏️ 如何编辑计划

### 方式一：页面内编辑（推荐）

1. 点击右上角 **「✏️ 编辑计划」** 按钮
2. 在弹出的编辑面板中：
   - **📅 日期设置**：选择起始日期、调整天数
   - **板块操作**：改名、换 emoji、选颜色（10 色可选）、删除
   - **任务操作**：改名、点击日期方块勾选排期、删除
   - **添加**：点击「+ 添加任务」或「+ 添加新板块」
3. 编辑完成后点击 **「✅ 保存并返回甘特图」**

> 每周换计划：编辑 → 改日期 → 增删调整任务 → 保存，搞定。

### 方式二：修改源码

用文本编辑器打开 HTML 文件，搜索以下注释标记：

- `修改日期 — 改这里` → `DAYS` 配置
- `修改分类颜色 — 改这里` → `CATEGORIES` 配置
- `修改任务模块 — 改这里` → `SECTIONS` 配置

## 💾 数据存储说明

| 数据 | 存储位置 | Key |
|------|---------|-----|
| 计划配置（板块、任务、日期） | localStorage | `gantt-cfg-v3` |
| 打卡记录 | localStorage | `gantt-done-v3` |

- 「↺ 重置打卡」只清空打卡记录，不影响计划配置
- 清除浏览器数据或换设备会重置所有内容
- 首次打开时使用内置的默认计划


如需部署为完整 PWA（支持离线 + 安装提示），可额外添加：

```

├── sw.js            ← Service Worker（离线缓存）
├── icon-192.png     ← App 图标
└── icon-512.png     ← App 图标
```
